Insert an element in an ArrayList instance.

Inserts a value at a given index of the ArrayList. If there is a value at the specified index, this value and its successors will be shifted right by one index. If capacity is to be exceeded, the data array will be resized.

Parameters
listThe ArrayList instance in which the element will be inserted.
iThe index at which the element will be inserted.

Returns
0 if successful, -1 if list is uninitialized.
